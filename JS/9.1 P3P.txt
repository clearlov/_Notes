P3P(Platform for Privacy Preferences): Cross Domain Cookie Reading


|- Set P3P Header by PHP -|
/* www.a.com */
header( 'P3P: CP="[P3P-HEADER-COMPACT-POLICY]"' );
setcookie('extinct', 100, time()+3600);

|- Set P3P Header by JS -|
xmlhttp.setRequestHeader( "P3P" , 'CP="[P3P-HEADER-COMPACT-POLICY]"' );

|- Set P3P Header by HTML -|
<meta http-equiv="P3P" content='CP="[P3P-HEADER-COMPACT-POLICY]"'>
